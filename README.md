1.Project Name: KenKeeper – Friendship Management Web App;

2.Description: KenKeeper is a modern web application designed to help users manage and maintain meaningful friendships. It allows users to track last contact dates, view friend details, and stay consistent with relationship goals through a clean and responsive interface


3.Technologies Used:
                     Next.js
                     React.js
                     Tailwind CSS
                     React Icons
                     JSON Data Handling (Local API)
                     Vercel (Deployment)


1.Friend Management System

Users can view a list of friends with profile pictures, tags, and status indicators (on-track, overdue, almost due).

2.Friendship Tracking

Tracks days since last contact and helps users maintain relationship goals with each friend.

3.Interactive UI with Navigation

Fully responsive UI with dynamic routing for:

Friend Details Page
Timeline Page
Stats Page
